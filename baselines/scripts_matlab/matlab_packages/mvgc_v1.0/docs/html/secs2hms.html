secs2hms

Convert time in seconds to hours/minutes/seconds

code

Syntax

   [h,m,s] = secs2hms(t)

Arguments

input

   t          time in seconds

output

   h          hours
   m          minutes
   s          seconds

Description

Return hours, minutes, seconds h, m, s of time t in seconds.
